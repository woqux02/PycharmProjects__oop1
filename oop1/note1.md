# 1.面向对象基础(ObjectOriented)
- OOP思想
    - 接触到一个任务,首先想到的是这个任务世界的构成
- 几个名词
    - OO:面向对象
    - OOA:面向对象的分析
    - OOD:面向对象的设计
    - OOI:xxx的实现
    - OOP:xxx的编程
    - OOA -> OOD -> OOI:面向对象实现过程
- 类和对象概念
    - 类:抽象名词,代表一个集合,共性的事物
    - 对象:具象的事物
    - 类和对象的关系
        - 一个具象,一个抽象
- 类中的内容
    - 表明事物的特征,叫属性(变量)
    - 表明事物的动作和功能,称为成员方法(函数)
